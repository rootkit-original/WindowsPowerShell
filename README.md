### Comandos Simplificados
```powershell
# Controle de serviços
compose-up          # Subir todos os serviços
compose-down        # Parar todos os serviços  
container-status    # Status completo

# Comandos diretos
podman ps           # Listar containers
podman images       # Listar imagens
```
